# 🏠 Machine Learning Based Real Estate Price Prediction

## 📌 Project Overview

This project predicts real estate property prices using Machine Learning algorithms. The system analyzes important property features such as **Square Feet area, Number of Bedrooms, Location, and Age of Property** to estimate the expected property price.

A **Flask-based web application** is developed where users can enter property details and get **real-time price predictions**.

---

## 🎯 Objective

The main objective of this project is to build and compare multiple regression models to identify the most accurate model for predicting real estate prices and deploy the model using a web application.

## 🤖 Machine Learning Models Implemented

The following regression models were implemented and compared:

* Linear Regression
* Random Forest Regressor
* XGBoost Regressor

The models were evaluated using the following metrics:

* Mean Squared Error (MSE)
* Mean Absolute Error (MAE)
* R² Score

Among these models, **XGBoost demonstrated better prediction performance compared to the other models.**

---

## 📊 Exploratory Data Analysis (EDA)

To understand the dataset and relationships between features, the following visualizations were created:

* Correlation Heatmap
* Price Distribution by Location (Box Plot)
* Square Feet vs Price Scatter Plot
* Market Trend Analysis

These visualizations help identify the most influential factors affecting property prices.
